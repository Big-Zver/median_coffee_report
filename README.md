## median_coffee_report

Консольная утилита для анализа трат студентов на кофе на основе CSV-отчетов. 
Проект написан на **Python** с использованием современных стандартов разработки (PEP 8, типизация, тесты).

## 🚀 Возможности
*   **Сбор данных**: Читает несколько CSV-файлов одновременно.
*   **Гибкий анализ**: Поддержка различных типов отчетов (сейчас доступен `median-coffee`).
*   **Красивый вывод**: Данные выводятся в виде аккуратных таблиц в терминале.
*   **Надежность**: Обработка ошибок (отсутствующие файлы, некорректные данные в таблицах).

## 🛠 Установка

1. **Клонируйте проект и перейдите в папку:**
   
       cd ~/PycharmProjects/median_coffee_report

2. Создайте и активируйте виртуальное окружение:

        python3 -m venv .venv
        source .venv/bin/activate

3. Установите зависимости:
   
        pip install -r requirements.txt


## 📈 Использование
Запустите программу, передав список файлов и тип отчета:

    python main.py --files math.csv physics.csv --report median-coffee



## Формат входных данных (CSV)

Файлы должны содержать заголовки: student, coffee_spent, mood и т.д.

Пример строки: Алексей Смирнов,2024-06-01,450,норм,Математика



## 🧪 Тестирование
Проект полностью покрыт тестами (Unit + Integration). Для запуска используйте:

    pytest -v
